DONUT CLICKER GAME

Fred The Baker Baking Company

The Donut Clicker is a project created by the instructors at We Can Code IT as a challenge project for students enrolled in the part time Spring Flex Boot Camp. \n The project is expected to get added to the students github repository.

Game Inspiration

The Cookie Clicker game was created in 2013, by French programmer Julien "Orteil" Thiennot. the user clicks on the a big cookie earning a single cookie per click. Users can spend their earned cookies on game assets.
